☆ Berry OS を Windows 領域へインストール

Berry OS は HDD に専用の領域を作ってインストールする方法の他、
既にインストールされている Windows NT/2000/XP の領域にインストールすることもできます。
加えて、USB メモリーへのインストールもサポートしています。

1. インストールに必要な要件

　・対応する Windows
    Windows NT/2000/XP など
 　　 →　Berry OS の起動に NTローダ(NTLDR) を利用するため 32 ビット NT 系 Windows が必要です。

　・HDD の空き領域
　　Windows NT/2000/XP の領域に Berry OS をインストールするためには、
　　640MB の HDD の空き領域が必要となります。

　　また、後述するデータ保存領域を作成する場合、さらに最低 100MB の 空き容量が必要です。

2. インストール先について

　　Windows がインストールされているドライブに Berry OS がインストールされます。
　　その際、以下のフォルダが作成されます。

　　\BERRY ディレクトリ

　　よって既に同名のフォルダが存在する場合は、別の名前に変更するか不要な場合は削除してください。

　　また、Berry OS インストーラは Windows のレジストリに書き込みを行いません。

3. インストール手順

　既存の Windows を通常どおり起動します。
　初めてインストールを行う場合、以下の手順で順番に作業を行います。

　3.1 Berry OS 本体

　1）Berry OS の CD を CD-ROM ドライブに挿入すると、自動的にインストーラが起動します。（図１）
　　　インストーラが起動しない場合は、Explorer 等で CD-ROM の内容を開き、Setup.exe をダブルクリックして起動します。
　2）Berry OS 本体をインストールします。「Berry OS のインストール」ボタンをクリックしてください。
　　　クリックできない場合は HDD の空き容量が不足しています。
　　　その場合、「終了」ボタンを押してインストーラを終了後、
　　　不要なファイルを削除して空き領域を作成してから再度インストールを行ってください。
　3）「Berry OS のインストール」ボタンをクリック後、使用許諾契約書が表示されます。（図２）
　　　内容をよくお読みになり、同意するのであれば「同意する」ボタンを、
　　　同意できないのであれば「同意できません」ボタンをクリックしてください。
　　　「同意する」ボタンを押した場合、インストール作業が続行されます。
　　　「同意できません」ボタンを押した場合、インストール作業が終了します。
　4）「フォルダ (Windows のドライブ名）:\BERRY は存在しません。作成しますか？」と確認されます（図３）ので、
　　　「はい」を押してください。
　5）ファイルのコピーが行われますのでしばらくお待ちください。（図４）
　6）コピーが完了すると入力待ちになります。（図５）

　3.2 Berry OS を OS 起動メニューに追加

　1）次に、Berry OS のブートローダを OS 起動メニューに設定します。
　　　「OS 起動メニューへ Berry OS を追加」ボタンを押してください。
　2）NTローダの設定ファイル Boot.ini （隠しシステムファイル）に Berry OS が追加されます。
　　　成功するとダイアログが出ます。（図７）
　　　「 OK 」ボタンを押すと入力待ちになります（図８）

　3.3 Berry OS のデータ保存領域の作成

　1）必要な場合は、データ保存領域を作成します。
　　　「Berry OS のデータ保存領域作成」ボタンを押してください。
　　　データ領域作成画面に移行します。（図９）
　2）容量を指定して「作成」ボタンを押すとデータ保存領域 ( \BERRY\berry.img ) が作成されます。
　　　既にデータ領域が存在する場合は上書きの確認画面（図１０）になりますので、
　　　既存のデータを消してもよい場合は「はい」を押してください。
　　　完了画面になります。(図１２)
　　　「 OK 」ボタンを押すと入力待ちになります（図８）

　3.4 Berry OS のスワップ領域の作成

　1）Berry OS を快適に使用したい場合は、スワップ領域を作成します。
　　　「Berry OS のスワップ領域作成」ボタンを押してください。
　　　スワップ領域作成画面に移行します。
　2）容量を指定して「作成」ボタンを押すとスワップ領域 ( \BERRY\swap.img ) が作成されます。
　　　既にスワップ領域が存在する場合は上書きの確認画面（図１０）になりますので、
　　　既存のデータを消してもよい場合は「はい」を押してください。
　　　完了画面になります。(図１２)
　3）これでインストール作業が全て完了しました。「終了」ボタンを押してインストーラを終了してください。

4. Berry OS の起動方法

　1）インストール完了直後の場合、一旦 Windows を終了します。
　2）既存の Windows を通常どおり起動します。
　3）Windows を起動するか Berry OS を起動するか選択メニューが表示されますので、
　　　Berry OSを選択します。

5. アンインストール手順

　既存の Windows を通常どおり起動します。
　次に Berry OS の CD を CD-ROM ドライブに挿入すると、自動的にインストーラが起動します。
　インストーラが起動しない場合は、Explorer 等で CD-ROM の内容を開き、Setup.exe をダブルクリックして起動します。

　5.1 Berry OS を OS 起動メニューから削除

　1）「OS 起動メニューから Berry OS を削除」ボタンをクリックしてください。
　2） NTローダの設定ファイル Boot.ini （隠しシステムファイル）から Berry OS を削除しています。
　3）削除が完了すると入力待ちになります。

　5.2 Berry OS 本体および Berry OS のデータ保存領域の削除

　データ保存領域も同時に削除されてしまいますので、データ保存領域内のデータが必要な場合は
　\BERRY\berry.img ファイルを別の場所へバックアップしてから以下の手順を行ってください。

　1）「Berry OS の削除」ボタンをクリックしてください。
　2）「フォルダ 'BERRY' とフォルダ内のすべてのファイルを削除しますか？」と確認されますので
　　　「はい」を押してください。
　3）削除が完了すると入力待ちになります。
